Explanation:
count_even_odd(numbers) function:

Takes a list of numbers as input.
Initializes even_count and odd_count to 0.
It iterates through each num in the numbers list.
Uses the modulo operator (%) to check if a number is even (num % 2 == 0) or odd.
Increments the respective counter.
Returns both even_count and odd_count.
Example Usage:

A sample list my_list is created.
The count_even_odd function is called with my_list.
The returned counts are stored in even and odd variables.
Finally, the counts are printed to the console.
